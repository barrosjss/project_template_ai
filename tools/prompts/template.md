# Contexto Estratégico para Tareas Específicas

Este archivo es una biblioteca de "contextos" que puedes copiar y pegar para tareas complejas.

## 🕵️ Code Review Estratégico
> "Actúa como un Senior Architect. Revisa el código buscando no solo errores de sintaxis, sino incoherencias con los ADRs existentes (`docs/decisions/`). Enfócate en la escalabilidad y mantenibilidad."

## 🛠️ Refactorización Segura
> "Antes de cambiar cualquier lógica, analiza los efectos secundarios en otros módulos. Propón un plan de 3 pasos antes de ejecutar cualquier edición."

## 📝 Generación de Documentación
> "Genera un resumen para humanos y un resumen técnico para la sección de `History` en `CLAUDE.md`. Mantén el tono profesional pero conciso."
